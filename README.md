# tech-news-headline-scraper
A Python-based web scraping project that collects the latest technology-related headlines from Hacker News and stores them in a text file.

## Technologies Used
- Python
- Requests
- BeautifulSoup

## Features
- Fetches live headlines
- Saves results with timestamps
- Simple and reusable code structure
